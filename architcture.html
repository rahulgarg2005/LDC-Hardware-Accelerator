First dense block. Takes two inputs: block_2_add (left path) and pre_dense_3 (right path, a projected 64-ch tensor). Each DenseLayer fuses these as 0.5×(new_features + x2) — a soft residual. This lets the network blend newly learned features with the projected right-path features, giving rich gradient flow for training.

SingleConvBlock(32→64, stride=2). Projects merged B2 output to 64 channels at 44×44. Added to block_3_down before entering Block 4. Bridges mid-level texture features into deeper layers.

SingleConvBlock(32→64, stride=1). Projects block_2_down to 64ch at 88×88. Serves as the stable "memory" input x2 in each DenseLayer of Block 3.

Combines block_3_down + pre_dense_2 (B2 resized). Then SingleConvBlock(64→96). This gives Block 4 context from both Block 2 and Block 3 simultaneously.

Attention-based fusion
🧠
CoFusion Block — Attention-weighted multi-scale fusion
Dynamically decides which scale's edge map is most reliable per pixel · not a blind average
4ch → 1ch

The four upsampled maps (out_1 to out_4) are concatenated into a B×4×352×352 tensor. CoFusion learns per-pixel attention weights using a Conv→GroupNorm→ReLU→Conv→Softmax pipeline. The Softmax ensures weights across the 4 scales sum to 1.0 at every pixel. The weighted sum collapses 4 channels to 1 — a single refined edge probability map.

Step 1 · Concat

torch.cat([out1,out2,out3,out4], dim=1) → B×4×352×352. All 4 scale predictions in one tensor.

Step 2 · Conv2d

4→32 channels, 3×3, pad=1. Learns cross-scale feature interactions.

Step 3 · GroupNorm + ReLU

GroupNorm(4 groups, 32ch) normalises. ReLU adds non-linearity. Stable across batch sizes.

Step 4 · Conv2d

32→4 channels, 3×3, pad=1. Projects back to 4 — one attention score per scale.

Step 5 · Softmax

dim=1 Softmax across 4 channels. At each pixel, weights sum to 1.0 — a proper probability distribution over scales.

Step 6 · Weighted sum

(x × attn).sum(1).unsqueeze(1) → B×1×352×352. Final fused edge map.

in → B×4×352×352 attn → B×4×352×352 out → B×1×352×352
Outputs
The model returns a list of 5 tensors — 4 intermediate scale maps + 1 fused. All are B×1×352×352. During inference the fused map is used; during training all 5 contribute to the loss (deep supervision).
